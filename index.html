Instrucciones para la Encuesta

Queremos que realices una simple encuesta con SurveyJS en dónde, además de enviar y recibir los datos de la encuesta, tendrás que cronometrar el tiempo que ha tardado en responder cada pregunta.

Los lenguajes de programación que deberás utilizar son:

  • HTML.
  • MySQL.
  • JavaScript con jQuery.
  • PHP.
  • JSON.

Empieza creando una encuesta en Create a Survey, Quiz, or Poll without Registration | Free Survey Tool (surveyjs.io), luego exporta el código JSON para crear una página web con jQuery Form Library | Getting Started Guide (surveyjs.io). Debes lograr que tu encuesta se imprima correctamente.

Las preguntas que puedes configurar para la encuesta (recuerda que es 1 pregunta por página) son:

  • Indica tu género (Hombre, Mujer, No-binario, Prefiero no contestar).
  • Edad (inserción numérica).
  • Una fotografía en la que aparezca una Pepsi y una Cocacola (Puedes juntar dos imágenes por Photoshop, ms-paint, o gimp).
  • ¿Prefieres la Pepsi o la Cocacola?

Continúa por crear una base de datos con una tabla en la que guardarás las respuestas de las encuestas enviadas por “personas encuestadas” y crea un fichero PHP que conecte la encuesta con la base de datos, recuerda que la encuesta envía una estructura de dato específica.
